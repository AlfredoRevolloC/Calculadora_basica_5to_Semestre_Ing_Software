# Calculadora Básica - Android

## Descripción
* Aplicación móvil Android desarrollada como parte de las actividad IndividualU3. Actividad de aprendizaje para RA: Actividad 5 de la asignatura PROGRAMACIÓN EN PLATAFORMAS MOVILES I de Ingeniería de Software.
* La aplicación implementa operaciones matemáticas básicas (suma) utilizando una interfaz de usuario funcional e intuitiva.

## Requisitos de la Actividad Cumplidos
* Diseño de la interfaz de usuario con Vistas y Componentes básicos (Botones y Etiquetas).
* Uso de Layouts (`LinearLayout`).
* Captura y manipulación de texto (`EditText` y `TextView`).
* Implementación de Eventos y Listeners (`setOnClickListener`).
* Manejo de notificaciones emergentes (`Toast`).

## Tecnologías Utilizadas
* **Lenguaje:** Java
* **Entorno de Desarrollo:** Android Studio
* **Control de Versiones:** Git / GitHub

## Instalación y Ejecución
1. Clonar el repositorio:
   `git clone https://github.com/AlfredoRevolloC/Calculadora_basica_5to_Semestre_Ing_Software.git`
2. Abrir el directorio del proyecto en Android Studio.
3. Esperar a que Gradle sincronice las dependencias.
4. Ejecutar el proyecto en un emulador virtual (AVD) o en un dispositivo físico conectado mediante depuración USB.
